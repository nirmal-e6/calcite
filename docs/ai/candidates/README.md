# Candidate Notes

## Purpose and scope

This directory is a quarantine area for short notes that might later become
reviewed knowledge. It should stay small and is healthy when empty.

## Why this matters

Most session notes should be deleted. This directory exists only to avoid
promoting raw notes into reviewed knowledge too early.

## Keep a note only when

- the note captures a potentially durable lesson
- the lesson is not yet generalized enough for `knowledge/`
- deleting it immediately would likely cause repeated rediscovery
